### Creation of Rails app
`rails new store`
* Note: Rails application generation can be customized by using flags. To see the options, `run rails new --help`.

## Rails differences with Django
* Rails handles the tracking of model changes and migration differently from Django.
  * Django track the changes in model through `python manage.py makemigrations` which creates the migration file containing the differences.
  * Rails directly creates the migration steps for the model change.
    The details can be found in [migrations](https://guides.rubyonrails.org/active_record_migrations.html) link.
  * Running of migration is the same with Django
